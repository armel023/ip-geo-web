# IP Geo Web – Frontend

A Next.js web application for IP geolocation lookup, user authentication, and history tracking.

## Features

- User authentication (NextAuth)
- IP geolocation lookup and map display (Leaflet, React-Leaflet)
- Search and view history of IP lookups
- Responsive UI with Tailwind CSS
- TypeScript support

## Getting Started

### Prerequisites

- Node.js (v18+ recommended)

### Setup

1. Navigate to this `frontend/` directory
2. Install dependencies:
   ```
   npm install
   ```
3. Start the development server:
   ```
   npm run dev
   ```

## Project Structure

- `app/` — Main Next.js app code
- `components/` — React UI components
- `public/` — Static assets
- `utils/` — Utility functions

## Environment Variables

- Configure API endpoints and authentication settings as needed in `.env.local`.
